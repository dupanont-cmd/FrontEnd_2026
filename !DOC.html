Algoritmos e Lógica de Programação

Um algoritmo é uma sequência de passos lógicos usada para resolver um problema.

Estrutura Básica

  • Entrada: Dados informados.
  • Processamento: Operações realizadas.
  • Saída: Resultado apresentado.

Exemplo

Calcular a média de duas notas:

  1. Receber nota 1
  2. Receber nota 2
  3. Calcular a média
  4. Mostrar o resultado
